# CRUD-REST-API Project

## Tecnologias utilizadas: Java, Spring Boot, Hibernate, MySQL

Projeto criado para colocar em prática o que aprendi sobre desenvolver uma REST API com as funcionalidades CRUD e também neste projeto implemento três técnicas de DAO: 

1. Usando EntityManager e API nativa do Hibernate; 
2. Usando EntityManager e JPA API padrão
3. Usando Spring Data JPA
